### 安装
安装环境
```
git clone git@github.com:bbaobelief/gecko.git
cd gecko
pip3 install -r requirements.txt 
```
### 创建
创建新爬虫
```
scrapy startproject 项目名称
```
### 运行
运行指定爬虫
```
cd 项目 && python run.py
```
